# AstronDaniel Portfolio

A professional, space-themed portfolio website for Daniel (AstronDaniel), showcasing projects, skills, and achievements.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: TypeScript

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Hero Section**: Immersive space-themed introduction.
- **About**: Bio and "Developer" class code snippet.
- **Skills**: Categorized list of technical skills.
- **Projects**: Showcase of GitHub repositories.
- **Contact**: Simple contact information.
- **Responsive Design**: Fully responsive for all devices.
